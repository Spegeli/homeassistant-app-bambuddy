# Home Assistant App: Bambuddy - BETA

![Supports aarch64 Architecture][aarch64-shield] ![Supports amd64 Architecture][amd64-shield]

This app integrates [BamBuddy](https://github.com/maziggy/bambuddy) into Home Assistant — a self-hosted print archive and management system for Bambu Lab 3D printers. It allows you to monitor and manage your prints directly from your Home Assistant instance.
